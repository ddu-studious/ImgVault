# ImgVault v2.5.0 - 图片整合功能 PRD

> 版本: v1.0  
> 日期: 2026-03-08  
> 优先级: P1

---

## 一、概述

### 1.1 背景
ImgVault 目前支持单图上传、存储、缩放、格式转换等能力。用户在实际使用中经常需要将多张图片和文字组合到一张图片上，生成海报、拼图、产品展示图等。

### 1.2 目标
在 ImgVault 中新增**图片整合 (Image Compose)** API，支持：
- 将多张图片按指定布局合成到一个画布上
- 在画布上叠加自定义文字（支持字体、颜色、大小）
- 内置常用模板（拼图、海报、产品卡）
- 输出结果自动存入 MinIO，可获取访问链接

### 1.3 用户价值
- 电商场景：快速生成产品组合图
- 内容运营：制作社交媒体海报
- 个人使用：图片拼接、添加文字注释

## 二、功能需求

### F1: 自由画布合成 API

**描述**: 用户提交合成指令（JSON），指定画布尺寸、背景色，以及多个图片/文字图层，后端按指令合成并返回结果。

**API 设计**:

```
POST /imgvault/api/v1/compose
Content-Type: application/json
```

**请求体**:
```json
{
  "canvas": {
    "width": 1920,
    "height": 1080,
    "backgroundColor": "#FFFFFF"
  },
  "layers": [
    {
      "type": "image",
      "imageId": 123,
      "x": 0,
      "y": 0,
      "width": 960,
      "height": 1080,
      "opacity": 1.0,
      "fit": "cover",
      "borderRadius": 0
    },
    {
      "type": "image",
      "imageId": 456,
      "x": 960,
      "y": 0,
      "width": 960,
      "height": 540,
      "opacity": 1.0,
      "fit": "cover"
    },
    {
      "type": "text",
      "content": "春季新品上市",
      "x": 100,
      "y": 900,
      "fontSize": 64,
      "fontFamily": "NotoSansCJK",
      "fontWeight": "bold",
      "color": "#333333",
      "maxWidth": 800,
      "lineHeight": 1.5,
      "textAlign": "left",
      "shadow": {
        "color": "rgba(0,0,0,0.3)",
        "offsetX": 2,
        "offsetY": 2,
        "blur": 4
      }
    }
  ],
  "output": {
    "format": "png",
    "quality": 90
  }
}
```

**响应**:
```json
{
  "code": 200,
  "data": {
    "imageId": 789,
    "imageUuid": "abc-123-...",
    "downloadUrl": "https://xxx/imgvault/storage/composed/2026/03/08/abc-123.png",
    "width": 1920,
    "height": 1080,
    "fileSize": 524288,
    "format": "png"
  }
}
```

**图层类型**:

| 类型 | 属性 | 说明 |
|------|------|------|
| `image` | imageId, imageUrl, x, y, width, height, opacity, fit, borderRadius, rotation | 图片图层 |
| `text` | content, x, y, fontSize, fontFamily, fontWeight, color, maxWidth, lineHeight, textAlign, shadow | 文字图层 |
| `shape` | shape(rect/circle/line), x, y, width, height, color, borderColor, borderWidth, opacity | 形状图层 |

### F2: 内置模板合成

**描述**: 提供常用合成模板，用户只需指定图片和文字，无需手动计算坐标。

**API 设计**:
```
POST /imgvault/api/v1/compose/template
```

**请求体**:
```json
{
  "template": "grid-2x2",
  "images": [123, 456, 789, 101],
  "text": {
    "title": "四季风光",
    "subtitle": "2026 精选"
  },
  "output": {
    "format": "jpeg",
    "quality": 85,
    "width": 1920
  }
}
```

**内置模板**:

| 模板 ID | 名称 | 说明 |
|---------|------|------|
| `grid-2x2` | 2×2 网格 | 4 张图等分排列 |
| `grid-3x3` | 3×3 网格 | 9 张图等分排列 |
| `grid-1-2` | 1+2 布局 | 左侧大图 + 右侧 2 张小图 |
| `grid-1-3` | 1+3 布局 | 顶部大图 + 底部 3 张小图 |
| `horizontal` | 横向拼接 | N 张图水平排列 |
| `vertical` | 纵向拼接 | N 张图垂直排列 |
| `poster` | 海报 | 背景图 + 标题 + 副标题 |
| `product-card` | 产品卡 | 产品图 + 名称 + 描述 + 价格 |
| `comparison` | 对比图 | 左右两图对比 + 标签 |
| `collage` | 自由拼贴 | N 张图随机角度拼贴 |

### F3: 模板预览

```
GET /imgvault/api/v1/compose/templates
```

返回所有可用模板的列表和预览缩略图。

### F4: 合成历史

合成结果自动存入 `img_image` 表，标记 `source_type = 'composed'`，可通过现有图片列表 API 查询。

## 三、非功能需求

| 维度 | 要求 |
|------|------|
| 性能 | 单次合成 P95 < 3s (1920×1080, 4 图) |
| 并发 | 支持同时 10 个合成任务 |

| 输出格式 | JPEG / PNG / WebP |

## 四、技术方案

### 4.1 架构

```
ComposeController (API层)
    │
ComposeAppService (应用层)
    │
    ├── TemplateEngine          # 模板引擎 - 将模板+参数转为图层列表
    ├── ImageComposeService     # 合成引擎 - Graphics2D 绘制
    │      ├── ImageLayerRenderer   # 图片图层渲染
    │      ├── TextLayerRenderer    # 文字图层渲染
    │      └── ShapeLayerRenderer   # 形状图层渲染
    ├── MinioStorageService     # 结果存储到 MinIO
    └── ImageRepository         # 记录入库
```

### 4.2 核心实现

- 使用 Java `BufferedImage` + `Graphics2D` 实现画布合成
- 图层按 `layers` 数组顺序从下到上绘制（后面的图层覆盖前面）
- 图片图层通过 `MinioStorageService` 从 MinIO 拉取源图
- 文字图层使用 `Font` + `FontMetrics` 计算布局，支持自动换行
- 合成结果写入 MinIO `composed/` 目录，记录到 `img_image` 表

### 4.3 字体方案

- 内置 Noto Sans CJK 字体文件 (约 16MB)
- 放置在 `resources/fonts/` 目录
- 启动时注册到 `GraphicsEnvironment`
- 支持 bold、italic 变体

## 六、验收标准

1. `POST /compose` 接口可正常合成多图+文字
2. 支持至少 4 种内置模板
3. 中文文字渲染正常（无乱码、无截断）
4. 合成结果自动存入 MinIO 并返回访问链接
5. 合成图片可在前端/Admin 端正常查看
